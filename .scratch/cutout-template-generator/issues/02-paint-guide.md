# Reduced Palette And Paint Guide

## What To Build

Extend the template pack so the maker gets a practical reduced palette and ranked budget craft paint matches from an editable paint catalog.

## Acceptance Criteria

- [ ] The app extracts a small adjustable reduced palette from the source image.
- [ ] Each palette color has 2-3 ranked budget craft paint matches.
- [ ] Paint matches include brand, paint name, approximate color swatch, and source note.
- [ ] The color guide page includes the palette and ranked matches.
- [ ] Tests cover deterministic palette extraction and ranking behavior.
